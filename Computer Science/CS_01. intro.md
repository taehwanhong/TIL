## 일반적인 PC의 구조

### 단어들

- Clock : 20Mhz 3Ghz 같은거 한 클락마다, 1개의 명령어를 수행할수 있다고 알려져 있음.
        : 컴퓨터가 일을 처리하는 단위.
        
        * C언어는 컴파일을 해서....Object파일이 되고, 여기에 Link를 연결해서 Executable이 되고 이건 binary file임. 이걸 실행하면 binary file을 메모리에 올려서 실행함.

- PC : program counter

- 메모리 : CPU계산하기 위해 컴파일된거 올라감

- 레지스터 : 값을 저장할수 잇는거. CPU랑 직접 통신

- 캐시 : 메모리랑 레지스터 사이


32bit머신은 명령어를 4byte 읽음. 이게 word임.
